# Gosleap

Only works on OSX

## Usage

![ryan gosling jumps across the screen](https://raw.github.com/koriroys/gosleap/master/gosleap.gif)

## Installation

Add this line to your application's Gemfile:

    gem 'gosleap'

And then execute:

    $ bundle

Or install it yourself:

    $ gem install gosleap
